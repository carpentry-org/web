serve

defn

(Fn [(Ref App a), (Ref String b), Int] ()) 

                        (serve app host port)

starts the web server on host:port.

Uses kqueue/epoll for concurrent connection handling with HTTP keep-alive. Sockets are non-blocking, reads accumulate into a per-connection buffer, and writes drain a per-connection buffer across writable events. A large response on one connection does not stall the event loop for others.

The loop is single-threaded, so peak throughput is bounded by one core. For multi-core scaling, run several copies behind a TCP load balancer.

static-dir

defn

(Fn [App, String] App) 

                        (static-dir app dir)

registers a wildcard GET route that serves files from dir on disk. The request path is appended to dir and read with Response.file.

The empty path serves dir/index.html. Paths containing .. segments return 404 to prevent directory traversal. Register static-dir after your API routes so it acts as a fallback.
